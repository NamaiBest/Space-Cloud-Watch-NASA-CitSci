How Classification Works

Each image is run through EfficientNet-B0, a convolutional neural network pretrained on ImageNet and fine-tuned on 890 sky images from three citizen science sources. The model outputs two things simultaneously:

Step 1 — NLC Detection

The binary head answers one question: are there noctilucent clouds in this image?

NLC Present Noctilucent clouds detected above the mesosphere (~80 km altitude), typically blue/silver in colour at twilight.
No NLC Ordinary clouds, clear sky, starfield, aurora, or any scene without noctilucent clouds.
Step 2 — NLC Type Classification

When NLC is detected, the type head identifies which of the four morphological types are present. Multiple types can appear in the same image.

Type 1 Veil Faint, featureless sheet — a thin, uniform luminous layer with no visible structure.
Type 2 Bands Long parallel streaks running roughly horizontally across the display.
Type 3 Waves Short, closely spaced wave-like undulations — ripples perpendicular to the bands.
Type 4 Whirls Partial or complete circular rings and swirl patterns caused by wind shear.
Confidence & Review Flag
≥ 85% High confidence Auto-accepted — still logged for audit
65 – 85% Medium confidence Flagged — human review recommended
< 65% Low confidence Priority review — model is uncertain

Entropy (a measure of prediction spread across classes) is also reported. High entropy (> 0.9) indicates the model may be seeing an out-of-distribution image — such as aurora, light pollution, or extreme lens flare.
